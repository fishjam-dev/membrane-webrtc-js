Official JS/TS client library for Membrane RTC Engine

Installation

Using npm:

npm install @jellyfish-dev/membrane-webrtc-js

or

yarn add @jellyfish-dev/membrane-webrtc-js

Using GitHub:

npm install jellyfish-dev/membrane-webrtc-js#<branch>
